# The Gibbs Invariant

This repository is dedicated to two connected theorems described as **The Gibbs Invariant**.

## Structure

1. `docs/gibbs_invariant/theorem_1_energy_invariant.md`
   - Statement-level version of Theorem 1 (error concentration invariant).
2. `docs/gibbs_invariant/theorem_1_technical_exposition.md`
   - Technical exposition and falsification criteria for Theorem 1.
3. `docs/gibbs_invariant/theorem_2_radius_invariant.md`
   - Statement-level version of Theorem 2 (circle-length budget invariant).
4. `docs/gibbs_invariant/theorem_2_technical_exposition.md`
   - Technical exposition and derivation for Theorem 2.

## Suggested Reading Order

1. Theorem 1 statement
2. Theorem 2 statement
3. Theorem 1 technical exposition
4. Theorem 2 technical exposition
